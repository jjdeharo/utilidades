Convertidor de fórmulas: Markdown IA a formato compatible

  1. Copiar el contenido del chat que contiene las fórmulas utilizando el botón que la IA tiene para copiar el contenido.
  2. Pegarlo más abajo, en el primer cuadro de texto.
  3. Descargar, con el botón "Guardar", el archivo (que tendrá extensión .md) con el texto y fórmulas. Alternativamente, también se puede copiar directamente el contenido generado y pegarlo en el botón "Paste" de Markdown Converter.
  4. Utilizar un servicio de Internet, como Markdown Converter para convertir este archivo en ODT (el formato de LibreOffice) o DOCX (el formato de Word). Para utilizar con Google Docs únicamente habrá que subir el archivo DOCX a Drive.

Más información en Conversión de texto y fórmulas generadas por ChatGPT a documentos de texto DOCX, ODT y Google Docs.
